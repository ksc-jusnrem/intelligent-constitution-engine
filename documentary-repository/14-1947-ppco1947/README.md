# Pakistan (Provisional Constitution) Order 1947

**Era:** Dominion of Pakistan — the Interim Framework (1947–1956)  
**Date:** 1947-08-14  
**Type:** order

## Dossier

Adapted the Government of India Act 1935 to a sovereign Pakistan: the colonial statute, with 'Governor-General' read for the Crown's machinery, became the first working constitution. Quaid-e-Azam Muhammad Ali Jinnah was sworn in as the first Governor-General the same day.

## Key provisions and effects

- 1935 Act (as adapted) + Independence Act = interim constitutional order
- Federal legislature: the Constituent Assembly doubling as ordinary legislature
- Federal Court of Pakistan continued the 1937 court's jurisdiction
- Governor-General retained wide powers — the fault line of the coming decade

## How it ended

Operated until the 1956 Constitution commenced on 23 March 1956.
